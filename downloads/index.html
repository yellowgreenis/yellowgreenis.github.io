Name Language Extension
time JavaScript .zip
temp Bash

- 현재 시간을 실시간으로 출력한다.

- .zip 내부에는 순수 JavaScript로 만든 파일과 jQuery로 만든 파일로 총 2개의 js파일이 있다.

- id="time 위치에 출력되며, 출력 위치는 // Output Point 부분에서 수정할 수 있다.

- 1초마다 출력되며, setTimeout()에서 숫자를 수정하면 반복 시간이 바뀐다.

- // 12 hour format 부분의 여러 줄 주석을 풀면 12시간 형식으로 출력된다.

- // Milliseconds 부분의 여러 줄 주석을 풀면 밀리초까지 출력한다.

- 요일 형식을 바꾸고 싶다면 var week = 부분을, 출력 순서를 바꾸고 싶다면 // HTML Code 부분을 수정하면 된다.

- 출력 결과는 여기서 볼 수 있다.

- 라즈베리 파이에서 CPU, GPU의 온도를 체크할 수 있는 Bash 스크립트 파일.

- sudo를 사용한다.

- 3초마다 출력되며, sleep 3에서 숫자를 수정하면 반복 시간이 바뀐다.

- 출력 결과는 홈 디렉토리에 temp.out이라는 이름으로 저장되며, 저장되는 위치와 이름을 바꾸고 싶다면 코드 내 ~/temp.out 부분을 수정하면 된다.

- 출력 예시 : 2021.02.31 12:34:56 Temperature | CPU : 00.0'C | GPU : 00.000'C
