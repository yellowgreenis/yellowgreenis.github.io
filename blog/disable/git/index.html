Git 최신 버전 설치

2021.08.30

Raspberry Pi OS 기준으로 작성했습니다.

Git

https://linuxize.com/post/how-to-install-git-on-raspberry-pi/
git config credential.helper store : 반영구적 암호 저장
git config credential.helper cache : 일정 시간 암호 저장 (기본 15분)
- git config credential.helper 'cache --timeout=3600' 3600초 연장
git config --unset credential.helper : 제거

위 모든 명령어에 --global 옵션을 주면 모든 프로젝트에 적용된다.
