한글 입력하기 (fcitx)

한글을 입력하기 위해서는 nabi, ibus, fcitx 등 여러 방법이 있는데, 다른 방법들 보다 fcitx가 제일 안정적인 것 같다.

Lite 버전에서 한글 입력할 사람은 없겠지? 그럼 그래픽 버전으로 설명한다. 사실 방법을 모른다..

1. sudo apt-get install -y fcitx fcitx-hangul
2. sudo vi /etc/default/im-config
3. IM_CONFIG_DEFAULT_MODE= 값을 fcitx로 수정.
4. im-config -n default
5. sudo reboot
6. fcitx-config-gtk3
7. 왼쪽 아래 +버튼 클릭
8. Only Show Current Language 체크 풀고 hangul 검색 후 선택 & OK.

+ 이제 컨트롤+스페이스바를 눌러서 언어를 변경할 수 있으며, 아래 창에서 단축키를 변경할 수도 있다.
