JAVA 16

아마 어느 리눅스에도 JAVA 16은 없을 거다.
그래서 Repository를 직접 추가하거나 파일을 구해서 설치해야 한다. Repo가 있긴 한가?

ARM 32Bit

라즈베리 파이 OS 32비트 버전도 여기에 포함된다.
1. wget https://github.com/AdoptOpenJDK/openjdk16-binaries/releases/download/jdk16u-2021-05-08-12-45/OpenJDK16U-jdk_arm_linux_hotspot_2021-05-08-12-45.tar.gz
2. sudo tar zxvf jdk-16.tar.gz : 압축을 풀고.
3. sudo mv jdk-16 /usr/lib/jvm/ : 압축을 풀어서 나온 디렉토리의 위치를 옮긴 후.
4. sudo update-alternatives --install /usr/bin/java java /usr/lib/jvm/jdk-16/bin/java 1을 입력한다.
- 3번에서 경로를 수정해도 상관은 없으나, 수정할 경우 4번에도 같이 수정해야 한다.
- java 외 javac, javadoc, javap 등 필요한 건 4번 명령어를 이용해서 등록하면 된다.

ARM 64Bit

64비트는 사용하지 않아서 모르겠다. 오라클에 arm 버전이 있긴 한데, 되는지 모르겠다.

RHEL/Fedora

CentOS를 32비트로 사용하는 사람은 없겠지?
그럼 오라클에서 'Linux x64 RPM Package'파일을 받으면 된다. 확장자는 .rpm
그리고 sudo rpm -ivh jdk-16.rpm
끝.

버전 변경하기

아래 명령어를 입력하면 자바가 설치된 경로와 그 왼쪽에 번호가 뜨는데, 사용하고 싶은 자바의 번호를 입력 후 엔터를 누르면 된다.
java 외 javac, javadoc, javap 등의 버전도 변경할 수 있다.

Ubuntu/Debian

sudo update-alternatices --config java

RHEL/Fedora

sudo alternatices --config java
